## Introduction

Frontend Mentor is a platform that provides web development challenges aimed at improving front-end coding skills. Each challenge comes with a design and a set of requirements that you must meet to complete the challenge. These challenges range in difficulty from beginner to advanced, making it a great platform for all levels of front-end developers to learn and improve their skills.

In this repository, you will find my solutions to some of the Frontend Mentor challenges. Each solution is complete with the code, along with a live demo where you can see the solution in action.

## Extra References

- [Josh Comeau CSS Global Reset](https://www.joshwcomeau.com/css/custom-css-reset/)

To improve CSS language and user experience, custom baseline styles can be used instead of outdated CSS Resets. A custom reset is unopinionated and can be used for any project.

- [Scott Ohara Screen Readers Only](https://www.scottohara.me/blog/2017/04/14/inclusively-hidden.html)

Visually hidden content adds important information for users of assistive technology, bridging the gap between sighted and low vision or blind users. This content is not meant to be visually accessible, except for exceptions like skip links or hidden until hovered/focused UI.

- [W3Schools HTML Semantic Elements](https://www.w3schools.com/html/html5_semantic_elements.asp)

A semantic element clearly describes its meaning to both the browser and the developer. It also improves accessibility and overall quality of products.
