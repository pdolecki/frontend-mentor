## Introduction

Frontend Mentor is a platform that provides web development challenges aimed at improving front-end coding skills. Each challenge comes with a design and a set of requirements that you must meet to complete the challenge. These challenges range in difficulty from beginner to advanced, making it a great platform for all levels of front-end developers to learn and improve their skills.

In this repository, you will find my solutions to some of the Frontend Mentor challenges. Each solution is complete with the code, along with a live demo where you can see the solution in action.
